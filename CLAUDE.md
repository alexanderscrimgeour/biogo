
This project simulates natural selection and genetic inheritence. 

- /grid package defines the world
- /jaro is a package for performing JaroWinklerSimilarity
- /ui is for creating a display of the world
- /simulation contains the code that performs the simulation

tests for packages should be written in a subdirectory /test
When writing code, unit tests should always be added for exported functions. 

Adhere to Google's Golang best practices and style guide.

When on windows, default to using Powershell instead of Bash commands.
